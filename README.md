## Focus

I work on parsers, editors, CLI tooling, and low-level software with a strong bias toward practical developer experience.

Current interests:

- Tree-sitter grammars and editor integrations
- Rust libraries and command-line tools
- Nix-based workflows and reproducible setups
- Small, sharp tools that remove friction from daily development
