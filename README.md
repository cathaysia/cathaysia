## Focus

I build low-level systems with an emphasis on performance, interoperability, and
pragmatic engineering.

Current interests:

- Software architecture and system design
- High-performance network programming
- Linux systems and performance work
- Kubernetes and cloud-native infrastructure
- Security engineering and applied cryptography
- FFI and cross-language integration
- Protocol design and transport security
- Noise-based systems, WireGuard, and Tor
- Tree-sitter grammars and editor integrations
- C++ and Rust libraries, services, and command-line tools
- Nix-based workflows and reproducible setups
- Small, sharp tools that remove friction from daily development
